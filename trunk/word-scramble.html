Objective

The point of this game is to discover all of the words made of the scrambled big word. It's pretty simple. Start typing, and hit 'enter' when you think you've got a word. If you mess up, 'backspace' will delete the last typed character. If you're out of ideas, hit 'enter' to mix up the characters, and maybe by chance it will spell a word you didn't think of. Currently, there is no database or 'back-end' to this, so there's just one word and it's matches to play with.

About

This game was a personal exercise by Matt Garrett at creating a Prototype and Scriptaculous powered JavaScript game. This was inspired by Text Twist and some other game with nearly identical gameplay. Actually, I had been wanting to create a word-based JavaScript powered game for a while now, and the Text Twist game seemed simple enough to start with.

Because this was a personal project, I have only tested this in Firefox. I don't intend on spending much time debugging other browser's problems.
